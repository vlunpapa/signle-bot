# GitHub 版本说明

## ⚠️ 重要提示

本版本已移除所有敏感信息，包括：
- 群组ID
- Bot Token
- API密钥
- 账号数据

## 🔒 隐私保护

- 所有 `.env` 文件已排除在版本控制之外
- Session文件（`.session`, `.session-journal`）已排除
- 数据库文件（`.db`, `.sqlite`）已排除
- 日志文件（`.log`）已排除

## 🚫 隐藏策略

以下策略在公开版本中已隐藏（代码保留但不在策略列表中显示）：
- **外源性爆发二段告警**：该策略的实现代码保留在 `src/strategies/engine.py` 中，但已从策略列表中移除

## 📝 使用前准备

1. 复制 `.env.example` 为 `.env`
2. 填写所有必需的环境变量
3. 确保所有敏感信息不会提交到Git仓库

## 🔐 安全检查清单

在提交代码前，请确认：
- [ ] 没有硬编码的API密钥
- [ ] 没有硬编码的群组ID
- [ ] 没有硬编码的Bot Token
- [ ] `.env` 文件已添加到 `.gitignore`
- [ ] 所有Session文件已排除
- [ ] 所有数据库文件已排除
